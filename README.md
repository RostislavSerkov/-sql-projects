# SQL Projects

A collection of SQL projects covering fundamentals through advanced database features, implemented on PostgreSQL.

## Projects

| Folder | Topic | Key Concepts |
|--------|-------|--------------|
| [B1-Basics](./B1-Basics) | SQL Basics | SELECT, WHERE, GROUP BY, HAVING, ORDER BY |
| [B2-Retrieving-data](./B2-Retrieving-data) | Retrieving Data | JOINs, subqueries, aggregation |
| [B3-Retrieving-data-advanced](./B3-Retrieving-data-advanced) | Advanced Retrieval | Window functions, CTEs, complex joins |
| [B4-DML](./B4-DML) | Data Manipulation | INSERT, UPDATE, DELETE, transactions |
| [B5-Snapshots](./B5-Snapshots) | Snapshots & Views | Views, materialized views, triggers |
| [B6-Salesman-problem](./B6-Salesman-problem) | Salesman Problem | Recursive CTEs, graph traversal in SQL |
| [B7-Database-index](./B7-Database-index) | Indexing | B-tree, hash, partial indexes, query optimization |
| [B8-Integrating-features](./B8-Integrating-features) | Advanced Features | Stored procedures, functions, advanced triggers |

## Tech Stack

- **Database**: PostgreSQL
- **Tools**: psql, pg_dump

## Skills Demonstrated

- Writing complex multi-table queries with various JOIN types
- Optimizing queries using indexes and execution plans (`EXPLAIN ANALYZE`)
- Implementing business logic with stored procedures and triggers
- Graph traversal using recursive CTEs (Salesman / TSP problem)
- Transaction management and data integrity
